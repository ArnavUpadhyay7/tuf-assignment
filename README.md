# Interactive Calendar

A clean, modern calendar UI built with React and Tailwind CSS, focused on smooth interactions, range selection, and polished visuals.

## ✨ Features

* Month navigation with smooth flip animation
* Date range selection (start → end) with hover preview
* Dynamic theme based on month images
* Holiday indicators with tooltip on hover
* Notes panel with auto-save feel
* Micro-interactions (hover + click animations)

## 🛠️ Tech Stack

* React
* Tailwind CSS
* Framer Motion

## 🚀 Getting Started

```bash
git clone <your-repo-link>
cd <project-folder>
npm install
npm run dev
```

## 🧠 Notes

* Focus was on **UI/UX and product feel**, not just functionality
* Designed to feel smooth, responsive, and visually consistent
* Theme system allows easy customization per month
